# Yeokie — Галерея

Личная галерея (фото и видео) в глубоких зелёных тонах.

**Сайт:** https://kannurali.github.io/Yeokie/

Статический сайт без сборки. Точка входа — `index.html`, ресурсы — в `assets/` (`photos/`, `videos/`).

## Нурри — осенняя страница с письмами

Отдельная вкладка «Нурри» в тёплой (коричнево-оранжевой) осенней палитре. Видна и доступна
только тем, кто вошёл по коду доступа (для гостей вкладка скрыта). Любой, кто знает код, может
писать письма, открывать и удалять их. Письма хранятся в общей базе (Firestore) с откатом на
`localStorage`. Конверт — алый, запечатанный сургучом; по клику печать ломается, клапан
откидывается, а письмо выезжает наружу для чтения.
